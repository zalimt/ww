Technical Information

  • Font Family: 'Poppins', sans-serif
  • Weights Available: 100, 200, 300, 400, 500, 600, 700, 800, 900
  • Styles: Normal and Italic for all weights
  • Display: swap (for better loading performance)
  • Fallbacks: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, sans-serif
